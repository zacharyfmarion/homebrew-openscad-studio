# Homebrew Tap for OpenSCAD Studio

This is the official Homebrew tap for [OpenSCAD Studio](https://github.com/zacharyfmarion/openscad-studio), a modern OpenSCAD editor with live preview and AI copilot.

## Installation

```bash
brew tap zacharyfmarion/openscad-studio
brew install --cask openscad-studio
```

## Updating

```bash
brew update
brew upgrade openscad-studio
```

## Uninstalling

```bash
brew uninstall openscad-studio
brew untap zacharyfmarion/openscad-studio
```

## About OpenSCAD Studio

OpenSCAD Studio is a cross-platform OpenSCAD editor featuring:

- Live 3D preview with instant re-renders
- AI copilot with Claude and GPT support
- Tree-sitter based code formatting
- Monaco editor with syntax highlighting
- 2D/SVG support for laser cutting designs

Learn more at the [main repository](https://github.com/zacharyfmarion/openscad-studio).
